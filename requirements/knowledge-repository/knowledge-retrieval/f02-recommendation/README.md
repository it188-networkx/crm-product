# Feature Definition - F-02 商机场景智能推送 (Opportunity-based Knowledge Recommendation)

## 特性概述 (Feature Overview)

**所属 Epic**：[T-04/E-02 知识检索与场景推送](../README.md)

**业务定义**：
本特性在售前打开商机详情页时，基于商机属性（行业、销售阶段、客户规模）自动推送与当前商机最相关的已发布知识单元，使售前无需离开商机详情页主动检索即可获得辅助材料，解决知识内容与商机详情页割裂、售前需来回切换应用获取支持材料的业务痛点。推送结果展示在商机详情页的专属扩展区域，不影响核心商机信息的正常展示。本特性聚焦"被动推送"场景，主动检索属 F-01，反馈提交属 F-03。

**范围边界**：

1. 包含：基于商机属性自动推送 Top-N 已发布知识单元、在商机详情页扩展区域展示推送摘要与版本信息、支持展开预览推送知识单元的完整内容。
2. 不包含：知识内容的创建与编辑（属 E-01）、主动检索功能（属 F-01）、反馈提交（属 F-03）、推荐相关性算法的训练与调整（由 E-03 负责）、商机详情页核心信息区域的修改（属 T-01/E-03）。
3. 参见 Epic [DEP-02]：本特性须集成嵌入 T-01/E-03 商机详情页提供的扩展区域，联调须在 GA 前完成。
4. 参见 Epic [DEP-03]：初期推送算法基于标签匹配和热度排序独立上线；E-03 向量化完成后作为相关性排序的渐进增强输入。
5. 参见 Epic [ASM-01]：假设 T-01/E-03 商机详情页已提供标准扩展能力，在本特性集成前完成稳定交付。
6. 参见 Epic [ASM-02]：初期推送算法基于标签匹配和热度排序，不依赖 E-03 向量化，可独立上线。

## 业务规则 (Business Rules)

- 参见 Epic [BR-01]（发布门控）：推送内容仅来自 E-01 状态为「已发布」的知识单元；归档、草稿、审核中状态的知识单元不得出现在推送列表中。
- 参见 Epic [BR-02]（权限过滤）：推送结果须按 GC-02 数据权限实时过滤；客户专属案例仅对相关商机负责人可见。
- 参见 Epic [BR-03]（推送上限）：商机详情页推荐知识展示上限 N=5 条（可配置），推送区域不得影响核心商机信息的展示。
- 参见 Epic [BR-04]（版本可见）：推送结果每条记录须展示对应知识单元的版本号与发布时间。
- **[BR-01]属性匹配策略**：推送算法基于商机的行业、销售阶段、客户规模标签进行匹配；当商机属性不完整时，基于已有属性进行匹配推送，不得因部分属性缺失导致推送完全失败；三项属性均缺失时，展示空推送提示（参见 BR-03）。[推导-待确认]
- **[BR-02]推送触发时机**：推送在售前打开商机详情页时自动触发，无需售前主动操作；同一商机在知识库内容或商机属性未发生变化时，多次打开详情页的推送结果保持一致。[推导-待确认]
- **[BR-03]空推送处理**：当无任何满足条件的已发布知识单元可推送时（含权限过滤后无结果），推送区域须展示「暂无相关推荐知识」，不得留白。[推导-待确认]

## 业务对象定义 (Business Object Definition)

### 推荐知识条目

推荐知识条目是在商机详情页扩展区域展示的每条自动推送记录，代表系统基于商机属性判断对售前当前商机跟进有辅助价值的已发布知识内容。

| 业务属性 | 业务说明 |
| :--- | :--- |
| 知识单元标识 | 唯一定位推送的知识单元，用于跳转详情与反馈关联 |
| 标题 | 知识单元的简短命名，用于快速识别推送内容主题 |
| 摘要 | 知识内容核心要点的简短描述，辅助售前判断与当前商机的相关性 |
| 知识类型 | 案例 / 话术 / 解决方案等类别，帮助售前快速理解内容性质 [推导-待确认] |
| 版本号 | 当前发布版本号，供售前确认推送内容是否为最新版本 |
| 发布时间 | 当前版本的发布时间，辅助售前评估内容时效性 |
| 推送依据 | 触发本次推送匹配的商机属性标签组合（行业 / 销售阶段 / 客户规模） [推导-待确认] |
| 排序位置 | 在 Top-N 推送列表中的排列次序，体现推荐优先级 |
| 所属商机 | 标识该推荐知识条目关联的商机，确保推送结果与商机上下文绑定 |

## 验收标准 (Acceptance Criteria)

| AC | 前置条件 | 触发动作 | 预期结果 | US Ref |
| :--- | :--- | :--- | :--- | :--- |
| AC-01 | 当前商机具有完整的行业、销售阶段、客户规模属性；存在已发布且符合权限的匹配知识单元 | 打开商机详情页 | 在商机详情页扩展区域自动展示最多 N=5 条相关知识推送；每条展示标题、摘要、版本号与发布时间；核心商机信息展示不受影响（参见 Epic BR-01, BR-03, BR-04） | US-01, US-02 |
| AC-02 | 推送列表中存在至少一条知识单元 | 对某条推送知识单元执行展开预览 | 展示该知识单元的完整内容，不跳转离开商机详情页 | US-03 |
| AC-03 | 当前商机属性对应的已发布知识单元权限过滤后无可推送内容 | 打开商机详情页 | 推送区域展示「暂无相关推荐知识」，不留白，核心商机信息展示不受影响（参见本地 BR-03） | US-01 |
| AC-04 | 当前商机行业属性缺失，销售阶段与客户规模属性存在 | 打开商机详情页 | 基于现有属性（销售阶段、客户规模）进行匹配推送；如有结果则展示，如无结果则展示「暂无相关推荐知识」（参见本地 BR-01） | US-01 |
| AC-05 | 某已推送知识单元随后被归档 | 下次打开同一商机详情页 | 该知识单元不再出现在推送列表中（参见 Epic BR-01） | US-01 |
