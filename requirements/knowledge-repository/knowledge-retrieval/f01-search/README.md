# Feature Definition - F-01 多维度知识检索 (Multi-dimensional Knowledge Search)

> 需求特性定义（A0304）｜所属主题：T-04 营销知识库
> **Feature 编号**：F-01 ｜ **优先级**：P0 ｜ **状态**：规划中
> 关联 Epic：[T-04/E-02 知识检索与场景推送](../README.md)

## 特性概述 (Feature Overview)

**所属 Epic**：[T-04/E-02 知识检索与场景推送](../README.md)

**业务定义**：
本特性为售前提供多维度知识检索能力，通过关键词与多维筛选器（客户类型 / 行业 / 竞品 / 销售阶段）快速定位匹配的已发布知识单元，解决售前在商机跟进中需要从多个系统手动搜索案例与话术、耗时 15～30 分钟的业务痛点。检索结果按热度与匹配度排序，并展示版本号与发布时间，确保售前使用当前有效版本的知识内容。本特性聚焦"主动检索"场景，不涵盖商机页面的自动推送（属 F-02）与反馈提交（属 F-03）。

**范围边界**：

1. 包含：关键词搜索、多维度筛选（客户类型 / 行业 / 竞品 / 销售阶段）、检索结果排序（热度 + 匹配度）、结果列表中版本号与发布时间的展示、知识单元详情查看。
2. 不包含：知识内容的创建与编辑（属 E-01）、商机详情页的自动推送（属 F-02）、反馈提交（属 F-03）、知识单元的下线与归档操作（属 E-01）。
3. 参见 Epic [DEP-01]：本特性只消费 E-01 状态为「已发布」的知识单元，须订阅 E-01 发布 / 归档事件以保证索引时效性。
4. 参见 Epic [DEP-03]：初期基于关键词与标签匹配独立上线；E-03 向量化完成后作为相关性排序的渐进增强输入，不影响本特性基础功能。

## 业务规则 (Business Rules)

- 参见 Epic [BR-01]（发布门控）：检索结果仅展示 E-01 状态为「已发布」的知识单元；归档、草稿、审核中状态的知识单元不得出现在结果中。
- 参见 Epic [BR-02]（权限过滤）：检索结果须按 GC-02 数据权限实时过滤；客户专属案例仅对相关商机负责人可见。
- 参见 Epic [BR-04]（版本可见）：检索结果每条记录须展示对应知识单元的版本号与发布时间。
- **[BR-01]筛选器组合逻辑**：各维度筛选条件之间为「与（AND）」关系；关键词与筛选器组合时同样采用「与（AND）」关系；未填写的筛选维度不纳入过滤条件，不视为空值匹配。[推导-待确认]
- **[BR-02]结果排序规则**：检索结果首先按匹配度降序排列；匹配度相同时按热度分降序排列；热度分由 F-03 反馈数据驱动更新（新知识单元热度分初始值待运营团队确认，TBD，默认分流：初始值 = 0）。[推导-待确认]
- **[BR-03]空结果处理**：当关键词与筛选条件组合无任何匹配的已发布知识单元时，须明确展示「无匹配的已发布知识单元」提示，不得展示空列表而无任何说明。[推导-待确认]

## 业务对象定义 (Business Object Definition)

### 知识单元检索结果条目

知识单元检索结果条目是售前在检索过程中看到的每条匹配记录，承载知识内容的核心摘要信息与排序依据，供售前快速判断内容相关性后决定是否查看详情。

| 业务属性 | 业务说明 |
| :--- | :--- |
| 知识单元标识 | 唯一定位该知识单元，用于跳转详情与反馈关联 |
| 标题 | 知识单元的简短命名，用于快速识别内容主题 |
| 摘要 | 知识内容核心要点的简短描述，辅助售前判断相关性 |
| 知识类型 | 案例 / 话术 / 解决方案等类别，帮助售前区分内容性质 [推导-待确认] |
| 版本号 | 当前发布版本号，供售前判断内容是否为最新版本 |
| 发布时间 | 当前版本的发布时间，辅助售前评估内容时效性 |
| 热度分 | 基于历史反馈累积计算的综合评分，作为同匹配度下的排序依据 |
| 匹配度 | 当次检索中与关键词和筛选条件的综合匹配程度 [推导-待确认] |
| 行业标签 | 知识单元适用的行业范围，用于多维筛选匹配 |
| 客户类型标签 | 适用的客户类型范围，用于多维筛选匹配 |
| 竞品标签 | 涉及的竞品名称，用于多维筛选匹配 |
| 销售阶段标签 | 适用的销售阶段，用于多维筛选匹配 |

## 验收标准 (Acceptance Criteria)

| AC | 前置条件 | 触发动作 | 预期结果 | US Ref |
| :--- | :--- | :--- | :--- | :--- |
| AC-01 | 有已发布且符合权限的知识单元；售前输入关键词并选择行业和销售阶段筛选条件 | 发起检索 | 返回同时满足关键词与所选筛选条件的已发布知识单元列表；结果按匹配度 + 热度排序；每条展示标题、摘要、版本号与发布时间（参见 Epic BR-01, BR-04；本地 BR-01, BR-02） | US-01, US-02 |
| AC-02 | 有已发布且符合权限的知识单元；售前仅输入关键词，不选择任何筛选条件 | 发起检索 | 返回关键词匹配的所有有权访问的已发布知识单元，不因筛选器未填写而被过滤（参见本地 BR-01） | US-01 |
| AC-03 | 当前关键词与所选筛选条件组合下无任何匹配的已发布知识单元 | 发起检索 | 展示「无匹配的已发布知识单元」提示，不展示归档或草稿内容（参见 Epic BR-01；本地 BR-03） | US-03 |
| AC-04 | 某知识单元为客户专属内容，当前用户不是相关商机负责人 | 发起检索 | 该知识单元不出现在当前用户的检索结果中（参见 Epic BR-02） | US-04 |
| AC-05 | 某知识单元已被归档 | 发起检索 | 该知识单元不出现在任何用户的检索结果中（参见 Epic BR-01） | US-01 |
